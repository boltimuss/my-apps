Copy and Paste Units

A quick way to replicate units in SimPlot is to use the Unit Copy/Paste feature. This will create a duplicate unit that has most of the attributes of the original unit.

The following attributes are copied from the original unit and pasted into a new unit:

  • Name;
  • Side;
  • Unit Class;
  • Unit Type;
  • Number;
  • Speed;
  • Course;
  • Range;
  • Altitude/Depth;
  • Sensors;
  • and Weapons.

A unit's perceptions, bearings, and active sensor status are not copied to the new unit as these are very specific to a given unit.

To COPY a unit, left-click a unit to copy and select the Units/Copy Unit menu item, or use CTRL+C. A message will be displayed in the upper left corner of the map indicating which unit is being copied.

To PASTE a unit, left-click a location for the new unit's position. Select the Units/Paste Unit menu item or use CTRL+V. The new unit (assigned a unique Track Number) with copied attributes will appear at the new location.

You may paste the new unit on top of the copied unit by copying and then pasting without clicking on the map (leave the copied unit still selected).

Use the ESC key to cancel the copy/paste function.
