dice v2.2 - RPG dice roller
********************************

dice v2.2 is a free RPG dice roller program.

Created in Microsoft Visual Basic 2010 Express

System requirements:

- Microsoft .NET Framework Version 2.0 *
- Windows XP 32/64bit, Windows Vista 32/64bit, Windows 7 32/64bit **
- Intel Pentium III 1 GHz or equivalent
- 64MB RAM
- 15MB hard drice space
- DirectX compatible sound card

* If you don't have Microsoft .NET Framework Version 2.0 download here:

http://www.microsoft.com/downloads/details.aspx?FamilyID=0856eacb-4362-4b0d-8edd-aab15c5e04f5&displaylang=en

** Maybe you can run it on Windows 98, Windows 2000, Windows Server 2003 32/64bit, Windows Server 2008 32/64bit, but these operating systems were not tested.
